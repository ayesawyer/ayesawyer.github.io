1.字符串最后一个单词的长度

题目描述

计算字符串最后一个单词的长度，单词以空格隔开。

输入描述：

一行字符串，非空，长度小于5000。

输出描述：

整数N，最后一个单词的长度。

示例：

输入：hello world
输出：5

编程

import java.lang.String;
import java.util.Scanner;

public class Main{

    public static void main(String[] args){

        Scanner sc = new Scanner(System.in);
        String words = sc.nextLine();
        String[] result = words.split(" ");
        int len = result.length;
        int len_result = result[len-1].length();
        System.out.println(len_result);

    }
}

讨论

该题目中值得注意的点在于：

  • 在用到Scanner/String及相关方法时别忘了import对应的包。
  • 获取数组的长度可利用属性length。
  • 获取字符串的长度要用到方法length()。

2.计算字符个数

题目描述

写出一个程序，接受一个由字母和数字组成的字符串，和一个字符，然后输出输入字符串中含有该字符的个数。不区分大小写。

输入描述：

第一行输入一个有字母和数字以及空格组成的字符串，第二行输入一个字符。

输出描述：

输出输入字符串中含有该字符的个数。

示例：

输入：ABCDEF/n A
输出：1

编程

import java.lang.String;
import java.util.Scanner;

public class Main{

    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        String aim = sc.nextLine();
        String word1 = sc.nextLine();
        String word2 = null;
        if(word1.compareTo("Z")>0){
            word2 = word1.toUpperCase();
        }else{
            word2 = word1.toLowerCase();
        }
        int count = getNum(aim,word1) + getNum(aim,word2);
        System.out.println(count);
    }

    public static int getNum(String str,String ch){
        String[] str1 = str.split(ch);
        int len = str1.length-1;
        return len;
    }
}

讨论

该题目中值得注意的点在于：

  • 大小写字符对应的ascii码值的范围及二者之间的转换（自己在做的时候考虑的是将目标字符分别转换为大小写进行判断，也可以考虑将目标字符串转换为大写或小写）。
  • 对于不同的解题思路的积累。

3.明明的随机数

题目描述

明明想在学校中请一些同学一起做一项问卷调查，为了实验的客观性，他先用计算机生成了N个1到1000之间的随机整数（N≤1000），对于其中重复的数字，只保留一个，把其余相同的数去掉，不同的数对应着不同的学生的学号。然后再把这些数从小到大排序，按照排好的顺序去找同学做调查。请你协助明明完成“去重”与“排序”的工作(同一个测试用例里可能会有多组数据，希望大家能正确处理)。

Input Param
n 输入随机数的个数
inputArray n个随机整数组成的数组

Return Value
OutputArray 输出处理后的随机整数

注：测试用例保证输入参数的正确性，答题者无需验证。测试用例不止一组。

输入描述：

输入多行，先输入随机整数的个数，再输入相应个数的整数

输出描述：

返回多行，处理后的结果

示例：

输入：11 10 20 40 32 67 40 20 89 300 400 15
输出：10 15 20 32 40 67 89 300 400

编程

import java.util.*;

public class Main{

public static void main(String[] args){
    Scanner sc = new Scanner(System.in);
    while(sc.hasNext()){                   //while循环实现对于多组输入数据的处理
        int len = sc.nextInt(); 
        int[] num = new int[len];
        for(int i=0;i<len;i++){
            num[i] = sc.nextInt();
        }
        int temp = 0;
        for(int m=0; m<len; m++){          //冒泡排序
            for(int n=0; n<len-m-1; n++){
                if(num[n]>num[n+1]){
                    temp = num[n];
                    num[n] = num[n+1];
                    num[n+1] = temp;
                }
            }
        }
        for(int j=0; j<len-1; j++){       //输出时判断是否存在重复数据
            if(num[j] != num[j+1]){
                System.out.println(num[j]);
            }
            if(j == len-2){
                System.out.println(num[len-1]);
            }
        }
    }
}
}

讨论

该题目中值得注意的点在于：

  • 注意对于输入数据的处理，需要进行循环处理。
  • 对于不同的解题思路的积累。

思路1——利用TreeSet这一数据结构，其中的数据都是按顺序存储的，并且作为Set，其中也不会包含重复数据。

4.字符串分隔

题目描述

  • 连续输入字符串，请按长度为8拆分每个字符串后输出到新的字符串数组；
  • 长度不是8整数倍的字符串请在后面补数字0，空字符串不处理。

输入描述：

连续输入字符串(输入2次,每个字符串长度小于100)

输出描述：

输出到长度为8的新字符串数组

示例：

输入：abc 123456789
输出：abc00000 12345678 90000000

编程

import java.util.Scanner;

public class Main{
public static void main(String[] args){
    Scanner sc = new Scanner(System.in);
    while(sc.hasNextLine()){
        String target = sc.nextLine();
        String sub = target;
        int len = target.length();
        if(len==0){                         //空字符串的情况
            continue;
        }
        for(int i=0; (i+8)<=len; ){
            String tar = sub.substring(i,i+8);
            System.out.println(tar);
            target = sub.substring(i+8);
            i=i+8;
        }
        int lengt = 8-target.length();
        for(int j=0; j<lengt; j++){
            target = target + "0";
        }
        if(!target.equals("00000000")){     //长度刚好是8的整数倍的情况
            System.out.println(target);
        }
    }
} 
}

讨论

该题目中值得注意的点在于：

  • 注意对于各种特殊情况的考虑。
  • 对于编程的简化。
